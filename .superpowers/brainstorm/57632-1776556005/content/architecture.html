📊 Data Flow:
1. Vendedor abre dashboard (React) → autenticado por tenant
2. Selecciona guión estático y abre en modal
3. Hace click en "Llamar" → Backend valida y llama Twilio API
4. Twilio ejecuta llamada → webhooks notifican cambios (inicio, fin, grabación)
5. Backend guarda eventos en PostgreSQL
6. Dashboard actualiza en tiempo real con estado de llamada
7. Analytics se actualizan con minutos, duraciones, etc.
